# Installation

## Prerequisites

- Python 3.x

## Steps

1. Clone the repository:
    ```bash
    git clone https://github.com/Zierax/LNK-payload-gen.git
    cd LNK-Payload-Generator
    ```

2. Install the required modules:
    ```bash
    pip install -r requirements.txt
    ```

## Module Installation

- For Windows:
    ```bash
    pip install pywin32
    ```

- For non-Windows:
    ```bash
    pip install pylnk
    ```
